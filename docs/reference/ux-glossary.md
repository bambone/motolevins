# Глоссарий UI (админки)

Единый словарь для **Консоли платформы** и **Кабинета клиента**.

- Подписи ролей: [`App\Filament\Support\RoleLabels`](../../app/Filament/Support/RoleLabels.php)
- Статусы доменов: [`TenantDomainStatusCopy`](../../app/Filament/Support/TenantDomainStatusCopy.php)
- Реестр настроек платформы: [`PlatformSettingRegistry`](../../app/Filament/Support/PlatformSettingRegistry.php)

## Продукт и панели

| Внутреннее | В UI |
|------------|------|
| Tenant | **Клиент** |
| Platform panel | **Консоль платформы** |
| Tenant admin | **Кабинет клиента** |
| Public site | **Сайт для посетителей** |
| Slug | **URL-идентификатор** |

## Роли платформы

| Системное | Подпись в UI |
|-----------|----------------|
| `platform_owner` | Владелец платформы |
| `platform_admin` | Администратор платформы |
| `support_manager` | Специалист поддержки |

## Роли в кабинете клиента

| Системное | Подпись |
|-----------|---------|
| `tenant_owner` | Владелец клиента |
| `tenant_admin` | Администратор клиента |
| `booking_manager` | Менеджер бронирований |
| `fleet_manager` | Менеджер парка |
| `content_manager` | Менеджер контента |
| `operator` | Оператор |

## Каталог и парк

| Концепция | В UI |
|-----------|------|
| Motorcycle (карточка каталога) | **Карточка в каталоге** / согласованная формулировка |
| Rental unit | **Единица парка** |

## Правила отображения

1. Внутренний числовой `id` не делать главной колонкой без нужды.
2. Raw JSON — в блоке «для разработчика» / с пояснением.
3. Роли и статусы — человекочитаемые строки из PHP-карт, согласованные с этим файлом.
