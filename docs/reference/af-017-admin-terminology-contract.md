# AF-017 — Контракт терминологии и UI-copy (админки)

Документ задаёт **правила**, по которым подписываются экраны **Консоли платформы** (`platform`) и **Кабинета клиента** (`admin`). Дополняет [ux-glossary.md](ux-glossary.md).

## Уровни языка

| Уровень | Где | Правило |
|--------|-----|--------|
| **Бизнес** | По умолчанию в кабинете клиента | Короткие русские подписи задач пользователя («Домен», «Заявка», «Следующий контакт»). Без внутренних имён полей БД и классов. |
| **Полупрофессиональный** | SEO, домены, аналитика, интеграции | Допустимы UTM, DNS, SSL, JSON-LD и т.п., если есть пояснение в `helperText` / описании секции или расшифровка в подписи («UTM (источник)»). |
| **Системный / advanced** | Консоль платформы, секции «инфраструктура», переопределения env | Допустимы технические детали (R2, режимы записи), но **не** голые английские заголовки без русского пояснения; опасные действия — с `modalDescription`. |

## Label, helper, tooltip

- **Label** — что за поле в терминах задачи (не имя атрибута в коде).
- **Helper text** — что это, когда менять, пример; критичный смысл не прятать только в tooltip.
- **Tooltip** — уточнение; для Filament: `hintIconTooltip` там, где уже есть иконка подсказки.

## Аббревиатуры

- Писать так: **«UTM (источник кампании)»** или подпись на русском + пояснение в скобках.
- Не оставлять в подписи колонки голые `utm_source` / `utm_medium` без расшифровки.

## Platform vs tenant

- В кабинете клиента избегать слова **tenant** в UI; использовать **«клиент»**, **«ваш сайт»**.
- В консоли платформы «клиент» = запись в `tenants`; технические поля (переопределение медиа) — с русскими заголовками и пояснением последствий.

## Единый словарь (канон)

Расширяемый список — в [ux-glossary.md](ux-glossary.md). Новые канонические пары добавлять туда и по возможности переиспользовать одни и те же формулировки в разных ресурсах.

## Инвентаризация изменений (итерации)

По мере доработок админки сюда или в отдельный журнал фиксируются заметные замены «до → после». См. раздел в конце [af-017-admin-copy-inventory.md](af-017-admin-copy-inventory.md).
